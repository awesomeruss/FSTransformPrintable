You need asp.net framework 4.5.1 on IIS
Copy the 'install' folder under wwwroot on your IIS server and rename to 'transformPrintable'
In IIS manager, create a new application pool called 'transformPrintable', then update the folder to be an application using this pool.

HTTP POST to https://...server.../transformPrintable/api/xslt
